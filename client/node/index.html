Creating a Project

  1. Initializing the project
    2. npm init

    This should then generate a package.json file that looks like this

    {
  "name": "project-name",
  "version": "1.0.0",
  "description": "Your project description",
  "main": "app.js",
  "devDependencies": {},
  "scripts": {
    "start": "node app.js"
  },
  "author": "Your Name",
  "license": "ISC",
  "dependencies": {
  }
}

    Obviously these values will be specific to your project based on the values you enter, and as you install dependencies, this file will automatically be updated
