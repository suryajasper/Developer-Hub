Node

Creating a Project

  1. Initializing the project

    2. npm init

    This should then generate a package.json file that looks like this

    {
  "name": "project-name",
  "version": "1.0.0",
  "description": "Your project description",
  "main": "app.js",
  "devDependencies": {},
  "scripts": {
    "start": "node app.js"
  },
  "author": "Your Name",
  "license": "ISC",
  "dependencies": {
  }
}

    Obviously these values will be specific to your project based on the values you enter, and as you install dependencies, this file will automatically be updated

  2. Installing dependencies

    3. To install dependencies, simply call npm install dependency-name or npm i dependency-name.
    This will automatically update your package.json and package-lock.json files accordingly.

    The main dependencies to install for a node.js project are express, http, and socket.io.

  3. Setting up the node.js file

    4. After you successfully set up your project and install the correct dependencies, you can start coding. The first thing to do is set up a node.js app with express to start handling client connections. Since you have already installed the http and express dependencies, we can write a few lines of code to set up our server

    var express = require('express'); // importing the express dependency
var app = express(); // initializing an app with express
var http = require('http').createServer(app); // creating the server
var io = require('socket.io')(http); // will be used later on for client-server communication
app.use(express.static(__dirname + '/client')); // give node.js access to your client folder. _dirname gets the current directory, and we are adding the link to the client folder.
var port = process.env.PORT || 3000; // this variable represents the port number, and in place of 3000, you can give it any number of your choice

http.listen(port, function(){
  console.log('listening on port ' + port.toString());
});

    As soon as we run this file with either node app.js or npm start, we can already connect to the server by going to the address 127.0.0.1/port or localhost/port on any browser.
